A modern, fully responsive portfolio website built with **PHP**, **SQLite**, **TailwindCSS**, and **Three.js** for an interactive 3D background. The site showcases my expertise in **ERP/Odoo**, **Website Development**, and **Mobile Apps**, with live project previews, a contact form that stores messages in SQLite and sends email notifications, and an animated GitHub showcase.

## ✨ Features

- **Dynamic pages** – Home, ERP/Odoo, Websites, Mobile Apps, Contact.
- **Interactive 3D background** – Particle system using Three.js.
- **Project showcase** – Cards with live screenshots (WordPress mshots) for each client website.
- **GitHub integration** – Displays repositories related to Odoo and Flutter (via manual cards, pulling from README content).
- **Contact form** – Stores messages in SQLite and sends email to `garomagalana3@gmail.com`.
- **Fully responsive** – Mobile‑first design with TailwindCSS.
- **Glassmorphism UI** – Modern cards, animated gradients, and hover effects.

## 🛠️ Tech Stack

- **Backend**: PHP 8.4, SQLite3
- **Frontend**: TailwindCSS, Alpine.js (for tabs), Three.js, Font Awesome
- **Email**: PHP `mail()` with spam‑reducing headers (or optional PHPMailer/SMTP)
- **Database**: SQLite (file‑based, no extra server)
- **Version Control**: Git + GitHub
